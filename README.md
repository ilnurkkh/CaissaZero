# CaissaZero

A custom chess engine built from scratch in modern C++23. 

This project is currently in active development. The goal is to build a high-performance, UCI-compliant engine utilizing bitboards, a custom move generator, and an Alpha-Beta search algorithm.

## Current Status: Phase 1
* Setting up core project structure and C++23 build tools.
* Designing fundamental data types and board representation.

## Build Instructions
This project is configured using GCC and standard Make/VS Code tasks.
Requires a compiler supporting `C++23` and `-march=native` hardware optimization.
